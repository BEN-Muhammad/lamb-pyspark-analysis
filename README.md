#  Analyse de la lutte sénégalaise (Lamb) avec PySpark

##  Description du projet

Ce projet consiste en une analyse de données liées à la lutte sénégalaise, appelée **Lamb**, un sport traditionnel très populaire au Sénégal.

L’objectif est de comprendre les dynamiques économiques autour de ce sport, notamment en analysant :
- le chiffre d’affaires
- les lutteurs les plus rentables
- les revenus par arène
- l’impact du nombre de tickets vendus

---

##  Technologies utilisées

- **PySpark** (traitement de données)
- **Python**
- **Matplotlib** (visualisation)
- **Google Colab** (environnement de développement)

---

##  Analyse et résultats

L’analyse exploratoire a permis de mettre en évidence plusieurs éléments importants :
- Une forte concentration des revenus autour de certains lutteurs majeurs
- Un rôle clé des grandes arènes dans la génération de revenus
- Une influence significative du nombre de spectateurs (tickets vendus)

 Toutes les étapes d’analyse, les visualisations ainsi que les interprétations détaillées sont disponibles dans :
- le notebook : `notebooks/lamb_analysis.ipynb`
- le rapport : `reports/rapport_lamb.pdf`

---
